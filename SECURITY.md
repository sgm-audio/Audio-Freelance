# Security

We take the security of this project seriously. If you believe you've found
a security vulnerability, please disclose it responsibly.

## Reporting a Vulnerability

**Do not open a public GitHub issue.** Instead, email:

**scott@sgmstudios.ca**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential mitigations you've identified

## Response

You'll receive an acknowledgment within 48 hours. We'll work on a fix
and coordinated disclosure timeline.

## Scope

- API key exposure
- Authentication/authorization bypass
- Remote code execution
- Data exfiltration
- Dependency vulnerabilities with known CVEs
